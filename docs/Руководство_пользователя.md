## Запуск приложения

Приложение состоит из двух частей. Запускать нужно по порядку.

### Шаг 1 — Запустить бэкенд (Java)

Открыть терминал и выполнить:

```bash
cd stud_practice/backend
mvn spring-boot:run
```

Дождаться строки:
```
Started BantumiApplication in X.XXX seconds
```

Сервер запустится на `http://localhost:8080`. Это окно терминала **не закрывать**.

### Шаг 2 — Запустить фронтенд (Python)

Открыть **второй** терминал:

```bash
cd stud_practice/frontend
pip3 install -r requirements.txt   # только при первом запуске
python3 main.py
```

Откроется окно игры.

## Как играть

### 1. Выберите количество камней

Радиокнопки «Камней в лунке» — выбор 3, 4, 5 или 6. Стандарт — 4. Выбор сразу начинает новую партию.

### 2. Делайте ходы по очереди

- **Игрок 1** кликает по лункам в **нижнем ряду**
- **Игрок 2** кликает по лункам в **верхнем ряду**
- Лунки противника и пустые лунки некликабельны
- Активные лунки слегка светлее — это подсказка

### 3. Следите за статусом

- Строка «Ход: Игрок X» показывает чья очередь
- После хода подсвечивается лунка, куда упал последний камень
- Все ходы фиксируются в журнале внизу

---

## Правила

| Ситуация | Что происходит |
|----------|---------------|
| Обычный ход | Камни раскладываются по одному против часовой стрелки. Калах противника всегда пропускается |
| **Бонусный ход** | Последний камень попал в ваш Калах — ходите ещё раз |
| **Захват** | Последний камень попал в вашу пустую лунку, напротив есть камни — все переходят в ваш Калах |
| Конец игры | У одного игрока закончились камни — каждый забирает оставшиеся свои в Калах |
| Победа | У кого в Калахе больше камней |

## Кнопки управления

| Кнопка | Действие |
|--------|---------|
| **Новая игра** | Сбросить доску и начать партию заново |
| **Отменить ход** | Откатить последний ход (недоступна при счётчике ходов = 0 и после конца игры) |

---

## Журнал ходов

Последние ходы отображаются снизу вверх:

| Запись | Значение |
|--------|---------|
| `#3  Игрок 1: лунка 2` | Обычный ход |
| `#4  Игрок 1: лунка 6 — бонусный ход!` | Игрок ходит снова |
| `#5  Игрок 2: лунка 3 — захват!` | Захват камней противника |

---

## Советы

1. **Бонусный ход** — считайте камни в лунке: если их ровно столько, сколько шагов до вашего Калаха, используйте её
2. **Захват** — держите пустые лунки напротив лунок противника с большим количеством камней
3. **Undo** — можно отменить неудачный ход и подумать ещё раз

## Частые вопросы

**Почему окно не открывается?**  
Убедитесь, что бэкенд запущен и в терминале есть строка `Started BantumiApplication`. Только после этого запускайте `python3 main.py`.

**`pip3: command not found`**  
Попробуйте `python3 -m pip install -r requirements.txt`.

**`mvn: command not found`**  
Установите Maven: `brew install maven` (macOS) или скачайте с maven.apache.org.
