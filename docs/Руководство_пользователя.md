## Запуск игры

Для запуска игры не требуется установка. Достаточно открыть файл `index.html` в любом современном браузере (Chrome, Firefox, Safari, Edge).

## Интерфейс игры

```
┌────────────────────────────────────────────┐
│              БАНТУМИ                       │
│  [Режим: Против ИИ] [Сложность: Средний]   │
│  [Камней: 4 ●────]                         │
├────────────────────────────────────────────┤
│           Ход: Игрок                       │
│                                            │
│  [Калах  ][12][11][10][ 9][ 8][ 7]         │
│  Игрок 2  ──────────────────────── Игрок 2  │
│  Игрок 1  ──────────────────────── Игрок 1  │
│            [ 0][ 1][ 2][ 3][ 4][ 5][Калах]  │
│                                            │
│  Счёт: Игрок 1: 0      Игрок 2/ИИ: 0      │
│  [Новая игра]  [Отменить ход]              │
└────────────────────────────────────────────┘
```

**Ваши лунки** — нижний ряд (Игрок 1). Кликайте по ним, чтобы сделать ход.

## Как играть

### 1. Настройте параметры перед игрой

- **Режим**: выберите «Против ИИ» или «Два игрока»
- **Сложность ИИ**: Лёгкий / Средний / Сложный
- **Камней в лунке**: от 3 до 6 (стандарт — 4)

Изменение любого параметра автоматически начинает новую партию.

### 2. Делайте ходы

- В режиме **«Против ИИ»**: кликайте по лункам в **нижнем ряду** (ваша сторона — Игрок 1)
- В режиме **«Два игрока»**: каждый игрок кликает по лункам на своей стороне в свою очередь

> Пустые лунки и лунки противника недоступны для клика.

### 3. Следите за статусом

- Надпись сверху показывает, чья сейчас очередь (красный = Игрок 1, синий = Игрок 2/ИИ)
- Когда ИИ думает, появляется анимированный индикатор «ИИ думает...»
- Жёлтая подсветка — лунка, куда упал последний камень
- Красная подсветка — лунка, с которой был произведён захват

## Правила

| Ситуация | Что происходит |
|----------|---------------|
| Обычный ход | Камни раскладываются по одному по лункам против часовой стрелки. Калах противника пропускается |
| **Бонусный ход** | Последний камень попал в ваш Калах → вы ходите ещё раз |
| **Захват** | Последний камень попал в вашу пустую лунку, а напротив есть камни → вы забираете все эти камни в свой Калах |
| Конец игры | У одного игрока закончились все камни → каждый забирает оставшиеся свои камни. Больше камней в Калахе = победа |

## Кнопки управления

| Кнопка | Действие |
|--------|---------|
| **Новая игра** | Начать партию заново с текущими настройками |
| **Отменить ход** | Вернуть последний ход назад (недоступно при конце игры) |
| **▼ Правила игры** | Раскрыть/скрыть краткие правила прямо на странице |

---

## Уровни сложности ИИ

| Уровень | Описание |
|---------|---------|
| Лёгкий | ИИ смотрит на 2 хода вперёд. Легко обыграть |
| Средний | ИИ смотрит на 5 ходов. Хорошая тренировка |
| Сложный | ИИ смотрит на 9 ходов с глубоким анализом. Сложно победить |

---

## Журнал ходов

В нижней части страницы отображается история ходов:
- Обычный ход: `#5 Игрок: лунка 3`
- Бонусный ход выделен **жёлтым**: `#6 Игрок: лунка 6 — бонусный ход!`
- Захват выделен **оранжевым**: `#7 ИИ: лунка 2 — захват!`

---

## Советы начинающему

1. **Следите за бонусными ходами**: если подсчитать камни в лунке и она «достаёт» до вашего Калаха — используйте это
2. **Планируйте захваты**: держите свои лунки пустыми напротив лунок с большим количеством камней противника
3. **Не торопитесь**: в Бантуми, как и в шахматах, промедление лучше, чем плохой ход
4. **Начните с уровня «Лёгкий»**: чтобы освоить правила, прежде чем бросать вызов сложному ИИ
