## 1. Назначение и цели проекта

Целью проекта является разработка программной реализации настольной игры «Бантуми» (Kalah / Манкала) для запуска в веб-браузере без установки. Игра поддерживает режим «один против компьютера» и «два игрока на одном устройстве».

---

## 2. Требования к функциональности

### 2.1 Игровая логика
| № | Требование |
|---|-----------|
| Ф-01 | Инициализация доски с N камнями в каждой из 12 лунок (N = 3–6, задаётся игроком) |
| Ф-02 | Раскладывание камней против часовой стрелки с пропуском Калаха противника |
| Ф-03 | Бонусный ход при попадании последнего камня в свой Калах |
| Ф-04 | Захват камней при попадании последнего камня в пустую лунку своей стороны |
| Ф-05 | Определение конца игры и подсчёт итогового счёта |
| Ф-06 | Отмена последнего хода (Undo) |

### 2.2 Пользовательский интерфейс
| № | Требование |
|---|-----------|
| Ф-07 | Визуальная доска с 12 лунками и 2 Калахами |
| Ф-08 | Отображение количества камней в реальном времени |
| Ф-09 | Подсветка последнего хода и захвата |
| Ф-10 | Индикатор текущего хода |
| Ф-11 | Журнал ходов с отметкой бонусных ходов и захватов |
| Ф-12 | Оверлей с результатом по окончании игры |

### 2.3 Игровые режимы
| № | Требование |
|---|-----------|
| Ф-13 | Режим «Против ИИ» с тремя уровнями сложности (лёгкий, средний, сложный) |
| Ф-14 | Режим «Два игрока» на одном устройстве |

### 2.4 Компьютерный противник (ИИ)
| № | Требование |
|---|-----------|
| Ф-15 | Алгоритм Минимакс с альфа-бета отсечением |
| Ф-16 | Глубина поиска: лёгкий — 2, средний — 5, сложный — 9 полуходов |
| Ф-17 | Эвристика: разность Калахов + бонусные ходы + потенциальные захваты |

## 3. Нефункциональные требования

| Категория | Требование |
|-----------|-----------|
| Совместимость | Chrome, Firefox, Safari, Edge — без установки |
| Производительность | Ответ ИИ (средний уровень) ≤ 1 секунды |
| Адаптивность | Экраны от 360px до 1920px |
| Надёжность | Состояние игры не теряется при нажатии кнопок |
| Читаемость | Код разделён на независимые модули с комментариями |
